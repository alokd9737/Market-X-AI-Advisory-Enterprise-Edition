# Market X AI Business Advisor

AI-powered consulting platform for business strategy, market expansion, route-to-market planning, growth assessment, and executive reporting.

## Features

- AI Strategy Advisor
- Growth Assessment
- Executive Reports
- Market Research
- Route-to-Market Consulting
- Consultation Management

## Tech Stack

- Streamlit
- Groq API
- Plotly
- ReportLab
- Scikit-learn

pip install -r requirements.txt
